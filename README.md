# mouse-presets
Wayland utility that allows to make custom mouse(or keyboard) presets.
This version currently in dev state.
If you want to change config, for example, mouse name, you should look to main.h

## Building
```bash
git clone https://github.com/Spalishe/mouse-presets
cd mouse-presets
make
```
Output program will be located in ./build folder

## Dependencies
This programs entirely relies on Wayland and wlr-layer-shell.
